Skill management
for developers

Manage your skills as an individual using the devradar editor or track competence management within your team by using both the editor and the devradar teams app.

  • manage skills via web UI
  • view all skills in a list or radar view
  • import/export via .toml files
  • overlay your teams devradars and focus on the key skills

Features

Devradar Editor

Document and present skills for individuals and teams. The devradar editor allows you to:

  • create your own techradar
  • customize adoption levels
  • define quadrant categories
  • create radar blips via GUI
  • view all skills in a list
  • search for specific skills
  • import/export via .toml files
  • share your devradar via URL

Devradar Teams (Preview)

Import your teams devradars and track progress in your team members competences.
